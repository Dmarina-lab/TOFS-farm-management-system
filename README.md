# ระบบจัดการฟาร์ม TOFS

## ภาพรวมโครงการ
ระบบจัดการฟาร์ม TOFS ถูกพัฒนาขึ้นเพื่อช่วยให้การจัดการฟาร์มเป็นไปอย่างมีประสิทธิภาพ โดยมีฟีเจอร์ในการติดตามข้อมูลการปลูก พื้นที่การเกษตร สถานะของพืชและสัตว์ ตลอดจนครบวงจรของกระบวนการผลิต

## ขั้นตอนการติดตั้ง
1. **ติดตั้ง Node.js**: ให้ดาวน์โหลดและติดตั้ง Node.js จาก [nodejs.org](https://nodejs.org/)
2. **Clone โครงการ**: ใช้คำสั่งต่อไปนี้ในการ clone โครงการ:  
   `git clone https://github.com/Dmarina-lab/TOFS-farm-management-system.git`
3. **ติดตั้ง Dependencies**: เข้าไปที่โฟลเดอร์ของโครงการและติดตั้ง dependencies โดยใช้คำสั่ง:  
   `npm install`
4. **รันโปรเจค**: ใช้คำสั่งต่อไปนี้เพื่อรันเซิร์ฟเวอร์:  
   `npm start`

ระบบจัดการฟาร์ม TOFS ช่วยให้การทำฟาร์มง่ายขึ้นและมีประสิทธิภาพมากขึ้น ด้วยการใช้เทคโนโลยีที่เหมาะสมและมีระบบจัดการที่ใช้งานง่าย.
